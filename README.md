# Instrumentation Amplifier PCB Design

📖 DESCRIPTION

This project presents the PCB design of an Instrumentation Amplifier developed using **KiCad**. The project includes the complete design process, from schematic capture to PCB layout, 3D PCB visualization, and Gerber file generation for PCB manufacturing.

 ✨ FEATURES

- Instrumentation Amplifier Schematic Design
- PCB Layout Design
- 3D PCB Visualization
- Optimized Component Placement
- PCB Routing
- Gerber File Generation
- Manufacturing-Ready PCB Design

🔩 COMPONENTS USED

- Instrumentation Amplifier IC
- Operational Amplifier (LM741)
- Resistors
- Pin Headers
- PCB Board

  TOOLS

- KiCad
- KiCad 3D Viewer


🎯 USE OF PROJECT

The Instrumentation Amplifier PCB is designed to amplify low-level differential signals while minimizing common-mode noise.

This type of circuit is commonly used in:

- Biomedical signal acquisition (ECG, EEG, EMG)
- Sensor signal conditioning
- Industrial measurement systems
- Data acquisition systems
- Precision analog instrumentation

This project also demonstrates the complete PCB design workflow using KiCad, making it a useful reference for students and beginners learning analog PCB design.
